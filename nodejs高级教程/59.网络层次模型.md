```js
// 网络层次模型
/* 
    // 无论采用那种模型，都是对通信过程的分层。每层中都存在许多协议，例如http属于应用层，tcp/udps属于传输层。
    1 OSI七层模型：
        -应用层：用户与网络的接口
        -表示层：数据加密、转换、压缩
        -会话层：控制网络连接的过程
        -传输层：控制数据传输可靠性（基于端口的协议层，所以数据在封装时必须要携带程序所占用的端口号。）
        -网络层：通过路由确定网络（ip协议：让我们依据ip地址确定源和目标网络）
        -数据链路层：确定了目标网络，直接进入到局域网内。想办法通过mac地址确定目标主机（常见协议有：arp寻址协议）
        -物理层：各种网络物理设备，定义了设备标准

    2 TCP/IP四层模型：(对七层模型的简化)
        -将应用层\表示层\会话层 合并成一个应用层
        -传输层 -> 传输层
        -将网络层 -> 主机层（网络层）
        -数据链路层\物理层 合并成一个接入层
*/
```
